DESCRIPTION
-----------
Short....
Generates Signature File ASCII-Art Letters. Windows.

Full....
SiggenPro 2D Generates ASCII-Art Letters for your signature file.
Includes 14 character sets. Many others available. Make your own 
using the Built-in character set editor or import DOS Figlets. 
A windows program.

SPECIAL REQUIREMENTS: Requires VBRUN300.DLL

INSTALLATION
------------
* In Windows run the unzipped SETUP.EXE.
* An ICON and group will be automatically created.

TO RUN - BASIC USAGE
--------------------
* Click on the SiggenPro Icon.
* Enter Text in 'Request' box and click on 'GENERATE' button.
* Choose 'Save As' to save the generated .SIG!
