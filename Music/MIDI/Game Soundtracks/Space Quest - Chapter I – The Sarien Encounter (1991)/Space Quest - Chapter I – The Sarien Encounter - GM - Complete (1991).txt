System Requirements:

- MIDI Playback Software capable of reading Type 1 Standard
   MIDI File format and embedded SysEx data
- General MIDI sound card or module  (Roland Sound Canvas SC-55 recommended)


This Standard MIDI File was recorded directly from Sierra's "Space
Quest 1" adventure game.  It has been recorded using Sierra's
MT-32 driver and converted to GM format. Note that MIDI files containing embedded SysEx data are not compatible with all MIDI file playback programs.
