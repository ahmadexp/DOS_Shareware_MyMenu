Inhaltsverzeichnis


1. Was hat diese Demo zu bieten?
Eine vollständige Stadt. Liberty City. Mehr als 2000 km Straßen zum 
Austoben. Vier Missionen. Hunderte Autos und Millionen Einwohner. Viel 
Spaß!

ABER ...
Das Spiel ist auf 6 Minuten begrenzt.

Tut uns leid!

Dafür gibt es aber drei verschiedene Versionen dieser GTA Demo. Eine 8 
BIT FARBEN Version, eine 24 BIT FARBEN Version und eine Version 
mit beiden Farbtiefen. Wenn Sie diese Demo von der Cover-CD einer 
Computerzeitschrift installiert haben, werden Sie sehr wahrscheinlich beide 
Versionen besitzen. Wenn Sie diese Demo aus dem Internet geladen haben, 
haben Sie sich beim Download für eine von beiden entschieden.


2. Systemanforderungen
Minimalvoraussetzungen: 486 DX4 / 100 MHz Prozessor + 16 Mb RAM 
+ 512Kb VESA kompatible SVGA Grafikkarte.

Empfohlenes System: 100MHz Pentium Prozessor + 16Mb RAM + 2Mb 
VESA kompatible SVGA Grafikkarte.


3. Installation
Starten Sie die Datei GTA.BAT von der CD-ROM. Auf ihr sind außerdem 
zwei Musikstücke enthalten. Wenn Sie diese während des Spieles hören 
möchten, so lassen Sie die CD bitte während des Startens im CD-Laufwerk.


4. Starten des Spieles
Geben Sie “GTA” im DOS ein.


GTA wird automatisch die UniVBE und Miles Treiber starten, um Ihre 
Grafik- und Soundkarte zu konfigurieren.


5.  Tastaturbelegung
Sie können im Spiel zwischen zwei verschiedenen Tastaturbelegungen 
auswählen, “Standard” und “Gängig”.

Funktionstasten
F1, F2,
F3, F4		:  Mit den Tasten F1 - F3 können Sie im 
Mehrspielermodus Nachrichten an Ihre Gegner senden. F4 sendet diese an 
alle Spieler.

F9		:  Stadtteilname anzeigen

ESC		:  Spiel verlassen
F11		:  Bildschirmmodus. Mit dieser Taste können Sie den am 
besten geeignetsten Modus für Ihren PC auswählen.


6.  Spielen von GTA.
- Du startest einsam und verlassen auf den rauhen Straßen von Liberty City. 
Du arbeitest für einen fetten, verschwitzten Bandenheini namens Bubby. Er 
wird Dich an einer der öffentlichen Telefonzellen anrufen und Dir nähere 
Infos zu Deinem Job geben.

- Um das nächste Telefon zu finden, folge einfach dem gelben Pfeil. Hast Du 
erst mal einen Auftrag, wird Dich der Pfeil immer zum nächsten Ziel führen.

- Um ein Gespräch anzunehmen, lauf einfach über das Telefon.

- Wenn Du eine Mission beendet hast, wird Dein Bonusmultiplikator erhöht. 
Je mehr Missionen Du erledigst, desto höher wird auch Deine Punktzahl.

- Die Kisten, die in der Stadt verteilt liegen, enthalten Bonusgegenstände. 
Um diese zu bekommen, zerstöre einfach die Kisten.

- Wenn Du ein Verbrechen begehst, wird dies kurze Zeit später der Polizei 
gemeldet. Je mehr Du vom rechten Weg abkommst, desto ‚beliebter‘ wirst 
Du auch für die Cops.

- Erkunde hunderte von Verstecken und Boni überall in der Stadt.
