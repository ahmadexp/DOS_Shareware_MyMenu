  To play Virtual Pool, unzip all files into a single directory, and type

        POOL <enter> to begin.



  The first time you start the game, the Setup Program will run, allowing you
  to set up the game for use on your system.

  You will then be presented with the StartUp/Help Menu.  Selecting (P) from
  this menu starts Virtual Pool.  Be sure to check out the other options
  on the StartUp menu for important information about Virtual Pool.

  Enjoy!
